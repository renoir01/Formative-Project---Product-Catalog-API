Product Catalog API

A powerful RESTful API for managing your product catalog

View API Documentation

Product Management

Create, read, update, and delete products with support for variants, categories, and tags.

Advanced Search

Search products by name, description, or tags. Filter by price range and category.

Inventory Tracking

Monitor stock levels, set low stock alerts, and manage product variants.

Key Endpoints

GET /api/products

List all products with filtering and pagination

POST /api/products

Create a new product

GET /api/products/stats

Get product statistics

GET /api/categories

List all categories

Example Request

GET /api/products?search=gaming&minPrice=50&maxPrice=200&category=electronics
